### v1.1 — Growth Features (Q4 2026)

- Referral system
- Multi-language support (i18n)
- Notification preferences
- Advanced analytics dashboard
- Export transactions (CSV/PDF)

### v1.2 — Monetization (Q1 2027)

- Affiliate program
- Tiered card products (premium, business)
- Bulk ordering for businesses
- Subscription management

### v2.0 — Enterprise (Q2 2027)

- SOC 2 Type II compliance
- Team accounts (multi-user)
- Approval workflows
- Custom integration API
- Dedicated support SLA

### v3.0 — Global (2028)

- Non-EVM chain support (Solana, Bitcoin L2s)
- Multi-currency fiat settlement
- Regulatory compliance (MiCA, MAS, NYDFS)
- Mobile native app (React Native)
