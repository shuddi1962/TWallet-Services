# Release Notes

## v1.0.0 — Core MVP (Target: Q3 2026)

### New
- Next.js 15 App Router application with RSC + Server Actions
- Authentication system (email OTP, register, login, password reset)
- WalletConnect v2 integration (5 wallets, 7 networks)
- Crypto payment verification via Edge Functions (Alchemy RPC)
- Physical and virtual card ordering
- Customer dashboard (cards, orders, transactions, profile)
- Admin dashboard (users, orders, payments, analytics)
- Landing page with SEO optimization
- RESTful API with OpenAPI 3.1 contracts
- Webhook system for external integrations

### Security
- RLS on all 19 database tables
- JWT-based session management
- Rate limiting on all API endpoints
- Security headers (CSP, HSTS, XFO)
- Audit logging for all sensitive operations

### Infrastructure
- Vercel production deployment
- Supabase (Postgres, Auth, Edge Functions, Storage, Realtime)
- GitHub Actions CI/CD pipeline
- Sentry error tracking
- Uptime monitoring
- Automated backups

### Known Issues
- See `KNOWN_LIMITATIONS.md`
