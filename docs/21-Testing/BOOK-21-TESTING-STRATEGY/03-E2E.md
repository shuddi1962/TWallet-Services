# E2E Tests

> **Framework:** Playwright | **Location:** `tests/e2e/`

## Critical User Flows

| Flow | Test File | Priority |
|------|-----------|----------|
| User registration → email verification → login → dashboard | `auth.spec.ts` | Critical |
| Browse cards → create order → see pending order | `ordering.spec.ts` | Critical |
| Connect wallet → see wallet in dashboard | `wallet.spec.ts` | Critical |
| Support ticket → create → reply → close | `support.spec.ts` | High |
| Admin dashboard → view users → suspend user | `admin.spec.ts` | Critical |
| Admin orders → filter → update status → assign tracking | `admin-orders.spec.ts` | Critical |
| Admin payments → view → flag → verify | `admin-payments.spec.ts` | Critical |

## Pattern

```ts
import { test, expect } from '@playwright/test';

test('user can register, verify, and log in', async ({ page }) => {
  await page.goto('/auth/register');
  await page.fill('[name="email"]', 'test@example.com');
  await page.fill('[name="password"]', 'SecurePass123');
  await page.fill('[name="full_name"]', 'John Doe');
  await page.check('[name="accept_terms"]');
  await page.click('button[type="submit"]');
  await expect(page.locator('text=Verify your email')).toBeVisible();

  // Simulate email verification (test helper)
  await verifyEmail('test@example.com');

  await page.goto('/auth/login');
  await page.fill('[name="email"]', 'test@example.com');
  await page.fill('[name="password"]', 'SecurePass123');
  await page.click('button[type="submit"]');
  await expect(page).toHaveURL(/\/app/);
});
```
