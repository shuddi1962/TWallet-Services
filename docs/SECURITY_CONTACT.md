# Security Contact

## Reporting a Vulnerability

If you discover a security vulnerability in TWallet Services, please report it privately.

**Email:** security@twalletservices.com

**Response SLA:** 24 hours (initial acknowledgment)

We follow a responsible disclosure process:

1. **Report** — Send details to security@twalletservices.com
2. **Acknowledgment** — We confirm receipt within 24 hours
3. **Assessment** — We triage and prioritize (target: 48 hours)
4. **Fix** — We develop, test, and deploy a fix
5. **Disclosure** — Coordinated public disclosure after fix is deployed

## Scope

- In-scope: TWallet Services web application, API, Edge Functions, infrastructure configuration
- Out-of-scope: Third-party services (Supabase, Vercel, Alchemy, WalletConnect), physical attacks, social engineering

## Rewards

We do not currently operate a bug bounty program but acknowledge all valid researchers in our security acknowledgments page.
