# 13 — Reports

> Component ID: ADM-013 | Status: Approved
> Exportable reports — operational, financial, analytical.

## Route

`/admin/reports` — Server Component

## Access

- Super Admin, Finance: full access (generate, export)
- Viewer: read-only, can download

## Report Types (7)

| Report | Description | Data Source | Export Formats |
|--------|-------------|-------------|----------------|
| Revenue Report | Daily/weekly/monthly revenue | payment_transactions | CSV, Excel, PDF |
| Order Summary | Order counts by status, time | card_orders | CSV, Excel |
| User Growth | New users over time | profiles | CSV, Excel, PDF |
| Transaction Volume | TX counts by network, token, day | payment_transactions | CSV, Excel |
| Payment Summary | Success/failure rates by network | payment_transactions | CSV, Excel, PDF |
| Card Product Stats | Orders per card product, revenue | card_orders + payment_transactions | CSV, Excel |
| Support Metrics | Tickets by status, priority, resolution time | support_tickets | CSV, Excel |

## Features

### Report Generator

| Field | Type | Description |
|-------|------|-------------|
| Report Type | Select | Choose from 7 types |
| Date Range | Preset + Custom | Today, This Week, This Month, This Quarter, This Year, Custom |
| Group By | Day/Week/Month | Aggregation period |
| Filters | Contextual per report | Network, token, status, etc. |
| Format | CSV / Excel / PDF | Output format |

### Report Preview

- Table preview of data before export
- Summary stats at top (total revenue, count, avg)
- Chart (if applicable)

### Generated Reports List

| Column | Description |
|--------|-------------|
| Name | Report name + type |
| Date Range | From - To |
| Generated By | Admin name |
| Format | Badge (CSV/Excel/PDF) |
| Size | File size |
| Generated | Date |
| Actions | Download, Delete |

## Server Actions

```ts
generateReport(data: ReportGenerationInput): Promise<{ reportId: string; url: string }>
downloadReport(reportId: string, format: 'csv' | 'excel' | 'pdf'): Promise<{ url: string }>
deleteReport(reportId: string): Promise<ActionResult>
```

## Supabase Query Pattern

```ts
async function generateRevenueReport(dateFrom: string, dateTo: string, groupBy: 'day' | 'week' | 'month') {
  const { data } = await supabase
    .rpc('generate_revenue_report', {
      date_from: dateFrom,
      date_to: dateTo,
      group_by: groupBy
    });
  return data;
}
```
