# Social Media

## Platform Strategy

| Platform | Purpose | Content Type | Frequency |
|----------|---------|--------------|-----------|
| X (Twitter) | Real-time updates, community, announcements | Short posts, threads, polls | 3–5x daily |
| LinkedIn | B2B, partnerships, thought leadership | Articles, company updates, insights | 3–5x weekly |
| Facebook | Broader audience, community building | Posts, events, groups | 1x daily |
| Instagram | Visual brand building, card showcase | Stories, reels, product photos | 3–5x weekly |
| Telegram | Direct community, support, alerts | Announcements, support, discussion | As needed |
| YouTube | Tutorials, deep dives | Video guides, product demos, webinars | 1–2x monthly |
| Discord | Developer community, power users | Discussion, support, beta access | Active daily |

## Content Strategy

| Day | Platform | Content |
|-----|----------|---------|

| Tuesday | LinkedIn | Industry insight or security tip |
| Wednesday | Instagram | Card showcase or user story |
| Thursday | X + Telegram | Product update or feature highlight |
| Friday | YouTube (bi-weekly) | Tutorial or demo video |
| Saturday | X | Community poll or Q&A |
| Sunday | — | Rest |

## Brand Voice

| Attribute | Description |
|-----------|-------------|
| Tone | Professional but approachable |
| Language | Clear, jargon-light (explain crypto terms) |
| Personality | Trustworthy, helpful, innovative |
| Values | Security, transparency, user-first |

## Social Media Management

For MVP, manage manually or use a scheduling tool:

1. **Scheduling:** Buffer / Hootsuite
2. **Analytics:** Native platform analytics
3. **Design:** Canva (pre-made templates)
4. **Monitoring:** Native notifications
