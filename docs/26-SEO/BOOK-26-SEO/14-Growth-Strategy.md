## Growth Loops

### Content → SEO → Traffic → Conversion

```
Blog post → Search ranking → Organic traffic → Signup → Card order
        ↕ (internal links to related content)
```

### Referral → Virality → Growth

```
User → Referral link → Friend signs up → User gets reward → Shares again
```

### Product → Engagement → Retention

```
Card active → Dashboard visit → Transaction → Notification → Return visit
```

## Phased Growth Plan

### Phase 1: Launch (Months 1–3)

- SEO fundamentals (metadata, sitemap, structured data)
- 4 blog posts
- Social media presence (X, LinkedIn)
- Launch announcement
- Community setup (Telegram, Discord)

### Phase 2: Build (Months 3–6)

- 8 blog posts per month
- Weekly newsletter
- Referral program launch
- Developer documentation
- First customer case study

### Phase 3: Scale (Months 6–12)

- 12 blog posts per month
- Affiliate program
- Paid acquisition testing
- Partnership program
- Community events (Twitter Spaces, AMAs)
