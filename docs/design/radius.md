# Border Radius

| Token | PX | Usage |
|-------|----|-------|
| `--radius-sm` | 8 | Buttons (small), inputs |
| `--radius-md` | 12 | Default buttons, cards |
| `--radius-lg` | 16 | Cards, modals, drawers |
| `--radius-xl` | 24 | Large cards, dialogs |
| `--radius-full` | 9999 | Badges, pills, avatars |

## Component Mapping

| Component | Radius |
|-----------|--------|
| Button (xs/sm) | `--radius-sm` (8px) |
| Button (md/lg/xl) | `--radius-md` (12px) |
| Card | `--radius-lg` (16px) |
| Dashboard card | `--radius-xl` (24px) |
| Wallet/Payment card | `--radius-xl` (24px) |
| Modal | `--radius-lg` (16px) |
| Drawer | `--radius-xl` (24px) top/bottom |
| Input | `--radius-sm` (8px) |
| Badge | `--radius-full` (9999px) |
| Avatar | `--radius-full` (9999px) |
