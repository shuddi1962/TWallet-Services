# Spacing

> 8px grid system. All spacing values in pixels.

## Scale

| Token | PX | REM | Usage |
|-------|----|-----|-------|
| `--space-0` | 0 | 0 | Reset |
| `--space-1` | 4 | 0.25rem | Micro spacing (icon gaps) |
| `--space-2` | 8 | 0.5rem | Tight spacing (stacked elements) |
| `--space-3` | 12 | 0.75rem | Small gaps |
| `--space-4` | 16 | 1rem | Default spacing |
| `--space-5` | 20 | 1.25rem | Medium spacing |
| `--space-6` | 24 | 1.5rem | Card padding |
| `--space-8` | 32 | 2rem | Section padding |
| `--space-10` | 40 | 2.5rem | Large gaps |
| `--space-12` | 48 | 3rem | Section margins |
| `--space-16` | 64 | 4rem | Page sections |
| `--space-20` | 80 | 5rem | Wide sections |
| `--space-24` | 96 | 6rem | Hero spacing |

## Usage

| Context | Token |
|---------|-------|
| Button padding | `--space-2` `--space-4` |
| Card padding | `--space-6` |
| Section padding | `--space-8` |
| Page margins | `--space-12` |
| Stack gap | `--space-4` |
| Grid gap | `--space-6` |
