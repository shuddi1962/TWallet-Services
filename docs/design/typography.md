# Typography

## Font Family

| Token | Value | Usage |
|-------|-------|-------|
| `--font-sans` | `Inter, system-ui, sans-serif` | Body, headings |
| `--font-mono` | `'JetBrains Mono', 'Fira Code', monospace` | Code, addresses, hashes |
