# Colors

> WCAG 2.1 AA minimum contrast ratio: 4.5:1 (normal text), 3:1 (large text)

| Token | Hex | Usage |
|-------|-----|-------|

| `--color-primary-hover` | `#1D4ED8` | Primary hover |
| `--color-primary-light` | `#EFF6FF` | Light backgrounds, badges |
| `--color-primary-dark` | `#1E40AF` | Dark backgrounds |
