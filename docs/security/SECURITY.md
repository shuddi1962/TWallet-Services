# Security Policy — TWallet Services

## Scope

This security policy applies to all systems, data, personnel, and third-party services that are part of the TWallet Services platform.

## Reporting Vulnerabilities

If you discover a security vulnerability, please:

1. **Do not** disclose it publicly
2. Email: `security@twallet.com`
3. Include a description, steps to reproduce, and affected version
4. We will acknowledge within 24 hours and provide an ETA for fix

## Responsible Disclosure

We follow a 90-day responsible disclosure window. We will:

- Acknowledge receipt within 24 hours
- Provide regular status updates
- Credit reporters (with permission) after fix is deployed
- Offer a bug bounty (TBD)
